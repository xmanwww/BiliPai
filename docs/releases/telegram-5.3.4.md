# BiliPai v5.3.4 更新说明（Telegram）

BiliPai v5.3.4 已发布，重点是首页推荐体验、竖屏稳定性和播放器细节优化。

## ✨ 新增功能
- 今日推荐单支持「收起 / 展开」。
- 推荐单支持「单独刷新」，不再强制和首页推荐同步刷新。
- 播放设置新增「播放顺序」五种模式：
  - 播完暂停
  - 顺序播放
  - 单个循环
  - 列表循环
  - 自动连播
- 横屏 / 竖屏播放器均可快捷切换播放顺序。

## 🛠 重点修复
- 修复竖屏滑到新视频后，进 UP 空间返回错回首条视频的问题。
- 修复竖屏合集续播偶发「有声音无画面 / 不能继续滑动」的问题。
- 修复详情页动作栏按钮列对齐问题（点赞列与其他列不齐）。
- 修复播放器内重复画质入口：左上角改为弹幕开关入口。
- 修复黄色乱码弹幕：过滤结构化指令 payload，仅显示可读文本。

## 🎬 图片预览体验更新
- 预览层改为全局 Overlay（非 Dialog），返回链路更稳定。
- 开关动画重构为更顺滑的 iOS 风格。
- 过渡中去除圆角变化抖动。
- 根据反馈，取消「可打断跟手返回」动画，统一为固定退场，避免中途卡住。
